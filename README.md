📊 Superstore Sales Dashboard | Tableau
Project Overview
This project presents an interactive Superstore Sales Dashboard built in Tableau using the Sample Superstore dataset from Microsoft Excel. The dashboard transforms raw sales data into meaningful visual insights, enabling users to analyze business performance across different regions, categories, customer segments, and time periods.
The project demonstrates data cleaning, visualization, and dashboard design techniques to support data-driven decision-making.
Objectives
Analyze overall sales and profit performance.
Identify top-performing and underperforming regions and categories.
Track sales trends over time.
Compare performance across customer segments.
Build an interactive dashboard for business analysis.
Dataset
Source: Sample Superstore Sales (Excel)
Records: Sales transactions containing customer, product, location, and order details.
Key Fields: Order Date, Ship Date, Sales, Profit, Quantity, Discount, Category, Sub-Category, Segment, Region, State, and Customer information.
Dashboard Features
📈 Sales and Profit Overview
🌍 Regional Performance Analysis
📅 Monthly Sales Trend
📦 Category & Sub-Category Analysis
🗺️ State-wise Sales Distribution
🎛️ Interactive Filters for Region, Category, Segment, and Date
Tools & Technologies
Tableau Desktop – Dashboard Development
Microsoft Excel – Data Source and Preparation
Key Insights
Identifies the highest-performing regions and product categories.
Highlights monthly sales and profit trends.
Enables comparison of sales across customer segments and states.
Supports interactive exploration through dynamic filters.
